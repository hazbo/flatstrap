The Grid System

Flatstrap uses a simple 12-wide grid system, this means you can stack as many 'columns' as you want as long as they add up to 12.

Simple Example

Below is a very simple example of how to create a grid.

	

		
This will span 3 "columns"

		
This will span 4 "columns"

		
This will span 5 "columns"

Multiple row example

What if we want to use multiple rows? No problem, it just works!

	
	

		
This will span 3 "columns"

		
This will span 4 "columns"

		
This will span 5 "columns"

	

	
	
	

		
This will span 3 "columns"

Offsetting columns

"What if I don't want the columns to sit directly next to each other?" I hear you ask! No problem, use the offset* class to push from the left.

It's worth noting that all offsets count towards your 12 grid limit, so take it into account.

	

		
This will span 3 "columns"

		
This will span 8 and be offset by 1 "column"
