Setup & Configuration

There's not a lot to set up, but if you're unfamiliar with LessCSS you may be a bit lost.

Configuration

Flatstrap comes with a robust configuration file full of lovely variables for you to tweak to get your website just right.

We've done this so you don't have to make crazy edits to the CSS behind Flatstrap's core to get your website perfect.

Setting up LESS

All the CSS within Flatstrap's core is pulled in through the base.less file, however your website will never read this LESS code. Instead we run the LESS compiler to turn all our lovely LESS into CSS, for more information on how to do this please visit the LessCSS website. If you're running a Mac I suggest using Simpless by the awesome guys at WeAreKiss.

Compilation

So you've made a change to your websites config? Or some of Flatstrap's core? Awesome! Now all you have to do is compile base.less. That's all! All the other LESS files are loaded in through the base less file, so there's no need to compile them too! (They'll be dragged into your final CSS file).
